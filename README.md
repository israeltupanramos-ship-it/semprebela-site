# Sempre Bela — Landing Page

Landing page estática para divulgação do Grupo VIP da Sempre Bela.

## Antes de publicar

1. Abra o arquivo `index.html`.
2. Procure por `https://chat.whatsapp.com/COLE-SEU-LINK-AQUI`.
3. Substitua todas as ocorrências pelo link real do grupo.
4. Depois envie os arquivos para o repositório do GitHub.

## Estrutura

- `index.html`
- `style.css`
- `script.js`
- pasta `assets`

## Meta Pixel

No `index.html`, procure pelo comentário:

`META PIXEL: cole aqui o código do Pixel quando tiver o ID`

Cole o código oficial do Pixel nesse local.

No `script.js`, existe um evento preparado para medir cliques no botão.
